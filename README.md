# Webu

## Image Providers

Webu supports:

- Unsplash
- Pexels
- Freepik

To enable them, add the provider API keys to `Install/.env`.

See [Documentation/image-providers-env.md](/Users/besikiekseulidze/web-development/webu/Documentation/image-providers-env.md) for the expected variables and backend-only handling rules.
